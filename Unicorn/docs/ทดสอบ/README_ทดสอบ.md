# โฟลเดอร์ทดสอบ — Unicorn / GTSAlpha-Forensics

โฟลเดอร์นี้เก็บชุดทดสอบและแบบขอผลการทดสอบ สำหรับ Lab หรือผู้รับผิดชอบทดสอบ

---

## ไฟล์ในโฟลเดอร์

| ไฟล์ | รายละเอียด |
|------|-------------|
| **ทดสอบระยะไกล_100_เช็ต.md** | ชุดตรวจ 100 รายการ — การทดสอบระยะไกล (Remote): การเชื่อมต่อ HQ–Agent, คำสั่ง GET_INFO / GPS_TRACK / FILE_PULL / REMOTE_SHELL, สถานะออนไลน์/ออฟไลน์ |
| **ทดสอบโซเชียล_100_เช็ต.md** | ชุดตรวจ 100 รายการ — การทดสอบโซเชียล: การดึง/แสดง/วิเคราะห์ข้อมูล Line, Facebook, Instagram, WhatsApp, ข้อความ, รูปภาพ, ติดต่อ |
| **แบบขอผลการทดสอบ.md** | เทมเพลตสำหรับขอและส่งผลการทดสอบ — วันที่, ผู้ทดสอบ, Lab, สรุปผ่าน/ไม่ผ่าน, หมายเหตุ |
| **รายงานผลการทดสอบ.md** | เทมเพลตรายงานผลสำหรับ Lab ส่งคืนหลังรันทดสอบ — สรุปผ่าน/ไม่ผ่าน, รายการที่ไม่ผ่าน, ไฟล์แนบ |

---

## วิธีใช้

1. **หา Lab / ผู้ทดสอบ** — ส่งไฟล์ชุดทดสอบ (100 เช็ต × 2) และแบบขอผลการทดสอบให้ Lab หรือผู้รู้
2. **รันทดสอบ** — ทำตามรายการตรวจในแต่ละไฟล์ บันทึกผลในคอลัมน์ ผลจริง / ผ่าน·ไม่ผ่าน
3. **ส่งผล** — กรอกแบบขอผลการทดสอบและแนบผล (หรือส่งไฟล์ที่กรอกแล้ว) กลับมาทีมพัฒนา
