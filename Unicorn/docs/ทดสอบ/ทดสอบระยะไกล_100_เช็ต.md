# ทดสอบระยะไกล — 100 เช็ต
## Unicorn / GTSAlpha-Forensics — Remote Testing

ชุดตรวจ 100 รายการ สำหรับการทดสอบระยะไกล: การเชื่อมต่อ HQ–Agent, คำสั่ง GET_INFO / GPS_TRACK / FILE_PULL / REMOTE_SHELL, สถานะออนไลน์/ออฟไลน์, ความปลอดภัยและ Audit

---

## วิธีใช้

- กรอกคอลัมน์ **ผลจริง** และ **ผ่าน/ไม่ผ่าน** ระหว่างหรือหลังทดสอบ
- ส่งผลกลับตามแบบขอผลการทดสอบ (แบบขอผลการทดสอบ.md)
