# เส้นทางเอกสาร - แชร์และเครื่องมือ ACESO

**อัปเดต:** ตามที่แชร์และแตกไฟล์ไว้แล้ว

---

## 1. แหล่งไฟล์ (Network Share)

| รายการ | เส้นทาง |
|--------|---------|
| **แชร์เอกสาร/ต้นทาง** | `\\Ihc-qutjv3999\d` |
| **ไฟล์ Installer บนแชร์** | `\\Ihc-qutjv3999\d\AcesoSetup.exe` (~2 GB) |

*เส้นทางแชร์นี้ใช้ copy ไฟล์ต้นทางได้ เมื่อเครือข่ายและสิทธิ์พร้อม*

---

## 2. เครื่องมือหลังแตกไฟล์ (Local)

| รายการ | เส้นทาง |
|--------|---------|
| **ไฟล์ Setup แตกแล้ว (เครื่องมือ)** | `D:\AcesoSetup.exe` |

หลัง copy จากแชร์มาไว้ที่ `D:\` แล้ว จะได้ **AcesoSetup.exe** ใช้รันติดตั้งหรือแตกเนื้อหาเพื่อเจอเครื่องมือ (ACESO.exe, Drivers, ID, ID_x64 ฯลฯ)

---

## 3. โครงสร้างหลังแตก/ติดตั้ง (อ้างอิง)

จาก INSTALLATION_SUMMARY โครงสร้างที่ได้หลังแตก AcesoSetup.exe จะมีลักษณะดังนี้:

- **ACESO.exe** – โปรแกรมหลัก
- **haspclnt.dll** – HASP License
- **ID/** – Drivers และ Resources
- **ID_x64/** – เครื่องมือ 64-bit (รวม PasscodeLookup.db)
- **TMP/** – Dependencies
- **WD/** – Working Directory
- **PF/** – Platform Files
- **RV/** – Report Viewer

---

## 4. การ copy จากแชร์มาที่ D:\ (ถ้ายังไม่มี)

ถ้ายังไม่มีไฟล์ที่ `D:\AcesoSetup.exe` ให้รันใน PowerShell (ยกเลิกการใช้งานไฟล์ที่ D:\ ก่อนถ้ามีโปรแกรมเปิดอยู่):

```powershell
Copy-Item -Path "\\Ihc-qutjv3999\d\AcesoSetup.exe" -Destination "D:\AcesoSetup.exe" -Force
```

---

## 5. การแตกไฟล์ AcesoSetup.exe เพื่อเจอเครื่องมือ

- **วิธีที่ 1:** รัน `D:\AcesoSetup.exe` เป็น Installer ตามขั้นตอนบนหน้าจอ แล้วเลือกตำแหน่งติดตั้ง
- **วิธีที่ 2:** ใช้ 7-Zip แตกเนื้อหา (ไม่รัน installer) – เปิด 7-Zip → เปิดไฟล์ `D:\AcesoSetup.exe` → Extract ไปโฟลเดอร์ที่ต้องการ จะเห็นไฟล์/โฟลเดอร์ภายใน (รวม AcesoFiles.tar.gz ถ้ามี)
