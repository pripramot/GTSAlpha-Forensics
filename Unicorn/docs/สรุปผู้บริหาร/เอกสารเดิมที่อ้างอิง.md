## วิธีใช้

- **นำเสนอผู้บริหาร:** ใช้ไฟล์ `ACESO_Unicorn_คุณสมบัติ_สรุปผู้บริหาร.md` (หรือ export เป็น .docx)
- **ขอเอกสารเดิมมาอ้างอิง:** ใช้รายการด้านบนระบุชื่อไฟล์และที่เก็บ แล้วขอจากทีมหรือ copy จาก path ที่ระบุ
- **จัดเก็บให้เรียบร้อย:** เก็บโฟลเดอร์ `สรุปผู้บริหาร/` ไว้ใน `Unicorn/docs/` เอกสารต้นฉบับอยู่ระดับเดียวกับโฟลเดอร์นี้ (`Unicorn/docs/*.md`)
