In order to proceed, the following screen must be present on the device:

If the RTGL Agent pop-up message is shown on the device, select 'OK'.

The correct date and time must be set on the device to ensure timestamp accuracy in the export. If the date and time is correct, proceed to the 'Instructions' tab above.

If the following image is present, then it is likely that the device has lost its date and time setting due to battery removal or its power was completely drained.

To correct the time and date on the device please do the following:

  • Select 'Adjust Date'
  • Switch off 'Automatic date & time' and then set the correct date and time manually
  • Select 'Back' which should take you back to the home screen
  • Re-open WhatsApp to display the list of message threads and proceed to the 'Instructions' tab above

If the 'date is inaccurate' message is still present on the device and its displaying the device's current date and time, then the WhatsApp version is out of date. Unfortunately it is not possible to extract data from an out of date WhatsApp version using this method. Press the 'Skip' button to continue the acquisition process.

There are different instructions for extracting WhatsApp depending on the version of the application running on the device. Follow the process below to determine what that version of WhatsApp is.

How to determine WhatsApp version

1. Select '︙' 2. Select 'Settings' 3. Select 'Help'




4. Select 'App info' 5. Make a note of the WhatsApp Messenger Version 6. Return back to the WhatsApp thread screen




Proceed to the 'Instructions' tab making sure to follow the relevant instructions for the identified WhatsApp version.

Follow the process below for each thread that is to be extracted.

WhatsApp versions pre 2.19.341 and post 2.20.11 - We recommend that each thread is extracted twice: once with media (if present) and then once without media.

WhatsApp versions 2.19.341 to 2.20.11 - Each thread only needs to be extracted once, with the 'with media' (if present) option.

Make sure at least one 1-2-1 message thread is extracted before proceeding (see 'Limitations' tab).

How to extract a thread

1. Select a message thread 2. Select '︙' 3. Select 'More'




4. Select 'Export chat' 5. If media is present in the thread this pop-up will appear. Select the relevant media export option referring to the note in bold above. 6. Select 'RTGL Agent'




Open the 'Status' tab above to view a list of files and threads extracted.

WhatsApp versions pre 2.19.341 and post 2.20.11

  • To identify sent messages, at least one 1-2-1 message thread sent from the device owner must be extracted. If the handset owner is not identified, all messages will be displayed as received.
  • Conversations exported with media files are limited to either the first 10,000 messages or 18MB of media file attachments. Conversations exceeding these limits will only contain the most recent messages.
  • Conversations exported without media files are limited to a maximum of 40,000 messages. When this is exceeded the oldest messages will be excluded.
  • If a conversation with media files exceeds the file attachment limit, then it is possible to acquire older messages by exporting the same conversation again but without media files. When viewing the conversation in the report, media files will only be present for the most recent messages. Messages with missing media files will be identified by a ‘<Media omitted>’ message in the report. Note that any message accompanying the media will also not be acquired.
    • Group Conversations
      • To determine whether a group conversation has been fully acquired, compare the creation timestamp with the first message timestamp.
    • 1-2-1 Conversations
      • To determine whether a 1-2-1 conversation has been fully acquired, compare the first message timestamp with the timestamp of the first message of the conversation on the device.
  • Threads that are acquired twice (with and without media) will be stitched together in the report with duplicate messages removed.
  • All message times are rounded to the nearest minute by WhatsApp.
  • Group chats in the Threaded Messages section of the Report display only the active participants that have sent messages.
  • It is not possible to differentiate between two group members that have the same name but different numbers as only the name is acquired.
  • Opening a message thread will change the status of unread messages.
  • Reliability of timestamps is dependent on the correct time being set on the handset during the time of export.

WhatsApp versions 2.19.341 to 2.20.11

  • To identify sent messages, at least one 1-2-1 message thread sent from the device owner must be extracted. If the handset owner is not identified, all messages will be displayed as received.
  • Conversations exported are limited to the first 40,000 messages.
  • All message times are rounded to the nearest minute by WhatsApp.
  • Group chats in the Threaded Messages section of the Report display only the active participants that have sent messages.
  • It is not possible to differentiate between two group members that have the same name but different numbers as only the name is acquired.
  • Opening a message thread will change the status of unread messages.
  • Reliability of timestamps is dependent on the correct time being set on the handset during the time of export.
